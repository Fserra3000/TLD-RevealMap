# TLD-RevealMap

## Requirements
MelonLoader

## Installation
1. Install MelonLoader(visit the Modding Discord for more info https://discord.gg/nHhyaPnr)
2. Drop RevealMap.dll into steamapps\common\TheLongDark\Mods

## What does it do?
Automatically reveals the map.
